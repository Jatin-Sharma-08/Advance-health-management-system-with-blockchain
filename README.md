It is a health management system project where I was solving the problem of data security by using Blockchain by creating a unique id of patients, designed to secure medical reports and provide a safe way of accessing the clinical data of patients for their caretakers and doctors using cryptography and decentralization.

The front end work will be operated by HTML, CSS, Javascript and backend work will be operated by NODE.JS, MySQL.

The Crypto currency used was Ether.

The software that I chose to run the project was Ganache.

The proposed system also maintains the balance between data accessibility and security. The proposed framework helps doctors, patients and caretakers to securely store and access patient’s medical data in EHR.

The project is designed to replace traditional paper-based medical records and to enhance hospital data management in healthcare sectors.
